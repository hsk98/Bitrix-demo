Pipelines & Sales Funnels

Residential · Commercial · Leasing

Tailored funnels for different lead types and transaction models. Stage-specific required fields keep data clean and reporting consistent.

Residential Sales

Owner/Buyer representation

  1. New Lead → auto-assign + SLA 30m
  2. Qualification → budget, location, beds, timeline
  3. Property Match → send curated shortlist
  4. Viewing Scheduled
  5. Offer & Negotiation
  6. Agreement/Contract
  7. Won / Lost (reason required)

Residential Leasing

  1. New Inquiry
  2. Qualification & Docs

Automation & SLA Design

Robots, triggers, and queues that reduce latency and enforce quality—platform-agnostic with Bitrix24-ready patterns.

Lead Capture & Routing

  • • Form/portal/webhook → Validate → Enrich (email/phone check)
  • • Dedupe against Contacts/Leads (fuzzy phone/email)
  • • Auto assign by team, source, geo, property class
  • • Start SLA timer: response due in 30m (residential), 1h (commercial)

Nurture & Follow-up

  • • No contact in 24h → create task + notify team lead
  • • Viewing scheduled → send SMS/iCal reminders to client & agent
  • • Offer created → auto-generate quote doc & checklist
  • • Idle 7 days → escalate and recycle to nurture list

Post-Close & Handoff

  • • On Won → tasks for compliance & payout
  • • Closed Lost → capture reason & trigger remarketing
  • • Lease expiry reminders at T-90/T-60/T-30
  • • Client satisfaction pulse (CSAT) 3 days after move-in

Example Bitrix24 Robots & Triggers

  • • Trigger: Incoming call/email → move to Qualification
  • • Robot @ New: Assign by queue; set SLA; send intro message
  • • Robot @ Viewing: Create reminder tasks; send calendar invites
  • • Robot @ Offer: Create quote; notify manager for approval
  • • Robot @ Idle: If no activity 3 days → escalate to team lead
  • • Robot @ Won: Create payout approval + finance notification
