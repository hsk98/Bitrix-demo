Pipelines & Sales Funnels

Residential · Commercial · Leasing

Stage-specific required fields keep data consistent and reporting clean.

Residential Sales

  1. New Lead → auto-assign + SLA 30m
  2. Qualification → budget, location, beds, timeline
  3. Property Match → send curated shortlist
  4. Viewing Scheduled
  5. Offer & Negotiation
  6. Agreement/Contract
  7. Won / Lost (reason required)

Residential Leasing

  1. New Inquiry
  2. Qualification & Docs
  3. Viewing
  4. Application & Screening
  5. Offer / Reservation
  6. Contract & Move-in
  7. Won / Lost

Commercial Sales

  1. Inbound / Prospect
  2. Qualification (use case, size, power, zoning)
  3. Viewing / Site Visit
  4. LOI / MOU
  5. Legal & Approvals
  6. Contract / Transfer
  7. Won / Lost

Commercial Leasing

  1. Inbound / Outreach
  2. Qualification (trade license, timeline, fit)
  3. Viewing / Technical
  4. Proposal & Negotiation
  5. HOA / Landlord Approvals
  6. Lease Execution
  7. Won / Lost

Stage-Required Fields

Qualification

  • Budget / Rent range
  • Bedrooms / Sqft / Power
  • Move-in timeline
  • Lead Type & Source

Viewing

  • Viewing date/time
  • Property ID
  • Agent assigned
  • Attendees

Offer

  • Offer amount / terms
  • Docs received
  • Decision-makers
  • Probability %

Closed

  • Reason Won/Lost
  • Commission
  • Marketing source
  • Cycle time (auto)
